# store-listing-optimizer

Turn your competitor's complaints into your competitive advantage.

Given your own app's current store listing and a competitor complaint brief (e.g., generated by `app-store-review-arbitrage`), this skill rewrites your listing to explicitly pre-empt competitor flaws and optimize for ASO (App Store Optimization).

## What it does

- **Audits** your current App Store or Google Play listing.
- **Cross-references** it against your competitor's broken promises and pain points.
- **Rewrites** your Title, Subtitle/Promotional Text, and Description to position you as the direct antidote to the competitor's flaws.
- **Enforces** strict character limits and ASO keyword best practices.

## Installation

```bash
npx "@opendirectory.dev/skills" install store-listing-optimizer --target claude
```

## Usage

1. Open your agent (e.g., Claude Code).
2. Ask it to run the skill:
   > "Use the store-listing-optimizer skill. Here is my current app description: [PASTE DESCRIPTION]. Here is the competitor brief from the app-store-review-arbitrage skill: [PASTE BRIEF]."

## Requirements

No API keys or third-party packages required. This is a pure LLM skill.
