# Output Format Reference

This file defines the exact Markdown structure the agent must produce for `devrel-backlog.md`.
The agent reads this file as part of its instruction set. Do not deviate from this structure.

## Field Definitions

| Field | Rules |
|---|---|
| `gap_title` | 4–10 words. States the missing doc clearly. No "How to" framing — phrase as the gap (e.g., "Middleware injection order not documented"). |
| `content_title` | A working title a human would write. Start with the topic, not "How to". |
| `priority_score` | Integer 0–100. Computed per scoring-guide.md. Never omit. |
| `frequency` | Count of distinct discussion threads in the cluster. |
| `engagement` | Sum of reaction_count + comment_count across all threads in the cluster. |
| `evidence_quote` | Verbatim text from a discussion body or comment. Max 3 sentences. Use blockquote format. |
| `source_url` | Full URL to the primary (most-engaged) discussion in the cluster. Never omit. |
| `content_type` | One of: `blog_post`, `tutorial`, `faq_entry`, `twitter_thread`, `video_script`. |
| `Draft FAQ / Doc Update` | A ready-to-use Markdown snippet containing the exact docs/FAQ addition needed to fix the gap. Must be based on the thread's resolution. |
| `Recommended Angle & Outline` | A specific framing and 3-4 point outline for the content asset, directly addressing the quoted confusion. |
| `why_this_matters` | One sentence. Explains founder value in plain language. Content section only. |
| `URGENT Badge` | `**⚠️ URGENT: Unresolved Community Pain**` inserted before the Evidence Quote if the cluster's unanswered rate is > 50%. |

---

## Anti-Patterns (Never Do These)

- ❌ "Here is a summary of your discussions..."
- ❌ Paraphrasing the evidence quote instead of quoting verbatim
- ❌ Omitting `source_url` from any item
- ❌ "This will improve your SEO / reach / engagement"
- ❌ "The community feels frustrated" (sentiment overclaim)
- ❌ "Suggested Action: Improve documentation" (You must draft the actual documentation!)
- ❌ Using words: delve, testament, comprehensive, leverage, seamless, in conclusion
